# Suchit Pathak | Data Analytics Portfolio

Welcome to my data analytics portfolio on GitHub! This repository serves as a central place to explore my projects and the technical blog posts where I explain my process, analysis, and findings.

My goal is to showcase my ability to tackle real-world data problems, utilize relevant tools and technologies, and communicate insights effectively.

You can view the main portfolio landing page by opening `Index.html` in your browser (or if this repo is hosted on GitHub Pages). The README below provides a direct overview and links to the individual projects and blogs contained within.

---

## 📁 Repository Structure

* `/`: The root directory containing this README, the main portfolio landing page (`Index.html`), and global styles (`style.css`).
* `/assets`: Contains images and other assets used across the portfolio and in the blog posts.
* `/projects`: This key directory holds the individual project-specific files, including the HTML files for each technical blog post and any direct project output files (like HTML dashboards).
